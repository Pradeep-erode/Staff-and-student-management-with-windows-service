CRUD operation on the Students/Management website

Business: Student mark entry website with separate Admin access, with an Excel Upload option that loads mark details for each student and their mark details, and schedules the Examination time, so that students will receive exam alerts before the examination.

Technologies: 
Web & API- .NET Core 5.0,
Windows Service - .NET Framework 4.7.2

Build as a Service-Repository Pattern and Dependency Injection for a loosely coupled architecture and to achieve Inversion of Control. With Restful Web API, FrontEnd and BackEnd were separated, and using Entity framework for Database Connectivity
